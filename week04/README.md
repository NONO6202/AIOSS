# Week 04: Teamwork and Collaborative Development

## 과제 내용

Feature 브랜치 전략으로 작업하고 Conventional Commits를 적용해 Pull Request를 생성한다. PR 리뷰 과정에서는 `[MUST]`, `[SHOULD]` 태그를 사용해 구조화된 피드백을 남긴다.

## 제출 범위

- [PR 리뷰 피드백 기록](REVIEW_FEEDBACK.md)
