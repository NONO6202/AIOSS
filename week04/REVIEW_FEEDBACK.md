# PR 리뷰 피드백 기록

## Review Tag Rule

| Tag | Meaning | Usage |
| --- | --- | --- |
| `[MUST]` | 병합 전 반드시 반영해야 하는 문제 | 요구사항 누락, 보안/대외비 노출 위험, 검증 불가 상태 |
| `[SHOULD]` | 병합 전 반영을 권장하는 개선 | 가독성, 유지보수성, 설명 보강 |
| `[NITS]` | 사소한 수정 제안 | 문구, 오탈자, 형식 정리 |
