# Rulebased Code Redacted

이 폴더는 ESG 지표산정 자동화 Agent의 rule-based 구현 구조를 공개 제출용으로 검열한 사본입니다.

원본 코드의 디렉터리 구조와 주요 모듈 경계는 유지했지만, 공개 저장소에 올리면 안 되는 산식, 임계값, 점수표, 원천 URL, 내부 문자열, 원본 데이터 흔적은 placeholder로 치환했습니다.

따라서 이 폴더의 코드는 실행 정확도나 재현성을 평가하기 위한 원본 코드가 아니라, 실제 프로젝트가 어떤 규모와 모듈 구조로 진행되는지 보여주기 위한 pseudocode 성격의 공개 자료입니다.

## Redaction Scope

- 문자열: `TXT_REDACTED`
- 원천 URL: `SRC_REDACTED`
- 숫자형 산식/임계값/날짜/점수: `1`, `2`, `3`, `4`
- 문자형 등급: `1`, `2`, `3`, `4`
- 주석: `# REDACTED`

## Purpose

이 사본은 실제 실행용 코드가 아니라 프로젝트 규모, 모듈 구성, rule-based 수집/계산 흐름을 증빙하기 위한 공개용 자료입니다. 공개 안전성을 위해 일부 문법적 완전성보다 민감정보 제거를 우선했습니다.

## Excluded

- 실제 ESG 산식과 가중치
- 세부 평가 조건
- 원본/정답 데이터
- 기업별 산정 결과
- 내부 API 키, 서버 경로, 캐시, 산출물
