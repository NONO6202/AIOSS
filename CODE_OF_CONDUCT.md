# Code of Conduct

## Our Pledge

이 저장소의 참여자는 공개 과제 저장소를 사용하는 모든 사람에게 존중과 책임 있는 협업 태도를 유지합니다.

## Expected Behavior

- 상대방을 존중하는 언어를 사용합니다.
- 문제 제기 시 사람보다 변경 사항과 근거에 집중합니다.
- 리뷰 피드백은 재현 가능한 문제, 개선 방향, 검토 기준을 포함해 작성합니다.
- 공개 저장소에 민감정보가 포함되지 않도록 주의합니다.

## Unacceptable Behavior

- 모욕, 괴롭힘, 차별적 표현
- 의도적인 허위 정보 공유
- 민감정보 또는 비공개 자료의 무단 공개
- 과제 평가나 협업을 방해하는 행위

## Reporting

문제가 발생하면 저장소 관리자에게 GitHub Issue 또는 Pull Request 코멘트로 상황을 설명합니다. 민감정보가 포함된 문제는 공개 Issue에 세부 내용을 작성하지 않고, 필요한 경우 비공개 채널에서 처리합니다.

## Enforcement

관리자는 문제의 심각도에 따라 댓글 수정 요청, 변경 요청, 기여 제한 등의 조치를 할 수 있습니다.
