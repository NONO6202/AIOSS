# 프로젝트 주제 제안서

| --- | --- |
| 프로젝트명 | ESG 지표산정 자동화 Agent |
| 개요 | 기업별 ESG/기업건전성 지표 산정 업무를 자동화하는 시스템 |
| 동기 | 장기간 수작업으로 진행되던 지표 산정 업무를 재현 가능하고 검증 가능한 파이프라인으로 전환하기 위함 |
| 예상 결과물 | 수집/계산/검증/리포트 작성 흐름을 갖춘 ESG 지표산정 자동화 도구 |

## 문제 정의

기존 기업건전성 지표산정 업무는 여러 기업의 공시, 재무, 인증, 제재, 소비자, 환경 관련 자료를 사람이 직접 확인하고 계산하는 방식으로 진행됐다. 이 과정은 많은 시간이 걸리고, 담당자 판단 차이와 반복 검증 비용이 크다.

전년도 수행에서는 LLM 중심 자동화 방식이 적용됐으나 요구사항을 충분히 만족하지 못했고 완성에도 이르지 못했다. 현재 팀은 해당 과제를 인계받아 rule-based 중심 구조와 Agent 보완 구조를 결합하는 방향으로 개선한다.

## 주요 기능

1. Rule-Based 수집 및 계산
   - 공개 가능한 원천 자료를 수집하고 지표 산정에 필요한 데이터로 정규화한다.
   - 계산은 명시적 규칙과 검증 가능한 절차를 기준으로 수행한다.

2. Agent 보완 수집
   - rule-based 수집으로 찾기 어려운 비정형 자료와 누락 근거 후보를 Agent가 보완 탐색한다.
   - Agent가 찾은 자료는 검토 대상으로 저장하며, 검증 없이 계산에 바로 반영하지 않는다.

3. 정답 검증 및 리포트 작성 Agent
   - 산출 결과와 기준 데이터를 비교해 오류 원인을 분석한다.
   - 검증된 결과를 바탕으로 기업별 리포트 초안을 작성한다.
