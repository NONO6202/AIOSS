# Label System

## Type

| Label | Purpose |
| --- | --- |
| `type:bug` | 결함, 누락, 오류 수정 |
| `type:feature` | 새 기능 또는 문서 개선 |
| `type:docs` | README, 보고서, 제출 문서 정리 |

## Priority

| Label | Purpose |
| --- | --- |
| `priority:P0` | 제출 품질이나 보안에 직접 영향 |
| `priority:P1` | 스프린트 내 우선 처리 |
| `priority:P2` | 일정 여유가 있는 개선 |

## Status

| Label | Purpose |
| --- | --- |
| `status:backlog` | 아직 착수하지 않은 후보 작업 |
| `status:to-do` | 이번 스프린트에 착수할 작업 |
| `status:in-progress` | 진행 중 |
| `status:review` | 검토 중 |
| `status:done` | 완료 |

## Sprint

| Label | Purpose |
| --- | --- |
| `sprint:1` | Sprint 1 대상 |
| `sprint:2` | Sprint 2 대상 |
